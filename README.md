# 📊 Customer Behavior Analysis

## 📌 Overview
This project demonstrates an end-to-end Data Analytics workflow, starting from data preprocessing to business intelligence reporting. The analysis focuses on understanding customer behavior, identifying purchasing patterns, and generating actionable insights through data analysis and visualization.

The project includes data loading and cleaning using Python, SQL analysis in MySQL, interactive dashboard creation in Power BI, and a final analytical report.

---

## 📂 Dataset
The dataset contains customer transaction and behavioral information, including demographic details, purchase history, product categories, spending patterns, and other relevant attributes used for analysis.

## 🔄 Project Workflow

### 1. Data Loading
- Imported the dataset into Python.
- Explored data structure and data types.

### 2. Data Cleaning
- Handled missing values.
- Removed duplicate records.
- Corrected inconsistent data.
- Converted columns to appropriate data types.

### 3. Exploratory Data Analysis (EDA)
- Analyzed customer demographics.
- Explored purchasing behavior.
- Identified trends and patterns.
- Visualized key insights using charts and graphs.

### 4. SQL Analysis
- Imported cleaned data into MySQL Server.
- Executed SQL queries to:
  - Filter and sort records
  - Aggregate customer metrics
  - Perform GROUP BY analysis
  - Calculate sales and customer statistics
  - Generate business insights

### 5. Power BI Dashboard
Developed an interactive dashboard to visualize:
- Customer Segmentation
- Sales Performance
- Revenue Trends
- Product Category Analysis
- Customer Purchase Behavior
- Key Performance Indicators (KPIs)

### 6. Report Creation
Prepared a report summarizing:
- Data cleaning process
- Exploratory analysis
- SQL findings
- Dashboard insights
- Business recommendations

## ▶️ How to Run

1. Clone this repository.
2. Install the required Python libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook and run all cells.
4. Import the cleaned dataset into MySQL Server.
5. Execute the SQL queries included in the project.
6. Open the Power BI (.pbix) file to explore the interactive dashboard.
7. Review the project report for detailed insights and recommendations.
